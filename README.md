# Shiva Spandana — Portfolio Website

Premium responsive portfolio in Ice Blue + Beige + Brown theme.

## Quick Start

```bash
cd portfolio
npm install
npm start
```

Open http://localhost:3000

## Build for Production
```bash
npm run build
```

## Sections
- Hero — Name, tagline, social links (GitHub, LinkedIn, LeetCode, Email)
- About — Summary, info grid, resume download button, social icons
- Skills — Tag-based skill groups from resume
- Projects — Career Guidance System, Role Based Mock Interview
- Education — SR University + Sri Chaitanya timeline + AICTE Internship
- Certificates — AWS, NVIDIA, AI Prompt Engineering
- Contact — All contact links + message form

## Customise
- Resume link: About.jsx → href on Download Resume button
- Social links: Hero.jsx and About.jsx
- Update projects, certs, etc. in respective component files
